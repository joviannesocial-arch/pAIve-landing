# pAIve Marketing Landing Page

The "Midnight Cyber-Premium" marketing site for pAIve, built with React, Vite, Tailwind CSS, and Framer Motion.

## Project Name
The project is configured as `paive-landing` in `package.json`.

## How to Open in PhpStorm
1. Open PhpStorm.
2. Select **Open** from the welcome screen or **File > Open**.
3. Navigate to and select the folder:
   `/Users/jovanne/.gemini/antigravity/playground/kinetic-granule`
   *(Note: The folder name might differ from the project name, but this is the correct directory).*
4. PhpStorm will automatically detect the `package.json` and configure the JavaScript environment.
5. Open the **Terminal** tab in PhpStorm (Alt+F12) to run commands.

## Setup & Running
1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Access the site at `http://localhost:5173` (or the port shown in terminal).

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS (v3 configuration)
- **Animations**: Framer Motion
- **Icons**: Lucide React
